## üöÄ Quick Start (3 Steps)

```powershell
# Step 1: Install System Software (manual)
# - Java 21 JDK from oracle.com
# - PostgreSQL 12+ from postgresql.org (create mindtrap database)
# - Node.js 18+ from nodejs.org

# Step 2: Install Project Dependencies
cd d:\mind-trap
npm install
cd backend
./mvnw clean install
cd ..

# Step 3: Run Application
.\start-all.ps1

# Visit: http://localhost:5173
```

**Total Time:** ~40 minutes (including downloads)

## ‚ú® Key Notes About This Project

### About PostgreSQL (Database)
‚úÖ **It's Local** - Runs on your machine, not in the cloud  
‚úÖ **PostgreSQL** - SQL database server  
‚úÖ **No Supabase** - Even though supabase/ folder exists, it's not used  
‚úÖ **Offline** - Works without internet  

**Note:** Supabase configuration files exist but the actual database uses **PostgreSQL running locally on your computer at localhost:5432**.

## üéì After Installation

Once everything is installed:

1. **Development:** Make code changes in `src/` or `backend/src/main/java/`
2. **Testing:** Use Swagger UI at http://localhost:8080/api/v1/swagger-ui.html
3. **Database:** Access via `psql -U postgres -d mindtrap`
4. **Frontend Testing:** Frontend auto-reloads at http://localhost:5173
