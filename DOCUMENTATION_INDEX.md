## üìã Installation Steps Overview

### The Basics (What You Need to Know)

This is a **React + Spring Boot + PostgreSQL** application.

**You must install (3 things):**
1. Java 21 JDK
2. PostgreSQL 12+
3. Node.js 18+

**Then run (2 commands):**
1. `npm install` (install frontend dependencies)
2. `./mvnw clean install` (install backend dependencies)

**Then start (1 command):**
1. `.\start-all.ps1` (start everything)

**Total time:** ~40 minutes
